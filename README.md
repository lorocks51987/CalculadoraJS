# 🧮 Calculadora Simples

Uma calculadora web desenvolvida com **HTML**, **CSS** e **JavaScript**, que realiza as quatro operações básicas: soma, subtração, multiplicação e divisão.

## 🚀 Funcionalidades
- Interface responsiva e moderna
- Operações básicas: +, -, *, /
- Botão de limpar (C)
- Botão de apagar (⌫)
- Visor interativo

## 🎯 Tecnologias usadas
- HTML
- CSS
- JavaScript
